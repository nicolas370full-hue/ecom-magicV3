# Ecom Magic — Generador de Imágenes de Marketing con IA

## Cómo subir a Vercel

1. Sube esta carpeta a GitHub:
   - Crea un repositorio nuevo en github.com
   - Sube todos los archivos

2. En Vercel:
   - Conecta tu repositorio de GitHub
   - Agrega la variable de entorno: FAL_KEY = tu_key_de_fal_ai
   - Deploy!
